# Tetris Lite
Basically a Tetris for PSP. Not done and may contain shitcode.

## How to build this?

1. Download and install [PSPDEV toolchain](pspdev.github.io/)
2. `cd build` and `cmake ../CMakeLists.txt`
3. `make`
